### [Job Tracker](https://github.com/ravalenr/job-tracker-api)
A full-stack application to track and manage job applications, statuses, and notes. Built with a production mindset — live deployed and fully containerised.
- **Stack:** Java · Spring Boot · React · PostgreSQL · Docker · JUnit/Mockito
- **Features:** Full CRUD, RESTful API, comprehensive test coverage
- **Live:** Deployed on Vercel (frontend) & Render (backend)

### [Full-Stack E-Commerce Platform](https://github.com/ravalenr/ecommerce-fullstack-project)
Complete client-server e-commerce solution engineered for high-traffic, production-grade environments.
- **Stack:** HTML5 · CSS3 · JavaScript (ES6+) · Node.js · Express.js · MySQL
- Dynamic shopping cart, real-time form validation, normalised DB schema (3NF), RESTful endpoints

### [Property Rental Management System](https://github.com/ravalenr/CA2_Integrated_Database)
Full-stack database application for managing property rentals, built with OOP principles and MVC architecture.
- **Stack:** Java · MySQL · JDBC · JUnit/Mockito
- 79% code coverage, normalised schema (3NF), full SDLC from requirements through deployment

- **Stack:** React · Python · Flask · Chart.js · Axios

### [Linux Server Networking Lab](https://github.com/ravalenr/linux-server-networking-lab)
